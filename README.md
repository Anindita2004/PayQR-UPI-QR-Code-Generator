# PayQR-UPI-QR-Code-Generator
PayQR is a simple and user-friendly desktop application built using Python and Tkinter that generates secure UPI payment QR codes instantly. The application allows users to enter UPI details such as UPI ID, recipient name, amount, and an optional note, and converts them into a scannable QR code for seamless digital payments.

*Features*
1.Generate UPI payment QR codes instantly
2.Input validation with error handling
3.Clean and responsive desktop GUI
4.Automatically saves generated QR as PNG
5.Customizable QR size and display resolution


*How It Works*

Enter the required payment details:
UPI ID
Recipient Name
Amount

then:
Click on Generate Secure QR
The application creates a UPI payment URL and converts it into a QR code
The QR code is displayed in the application and saved locally as upi_qr.png
